## About This Project
Firmware ini berbasis dari RTAWRT, dibuat untuk menjadikan sebuah stb sebagai perisai utama dalam berinternet dirumah, dilengkapi dengan Adguardhome sehingga menjadikan internet keluarga dengan aman dan sehat
## Feature
- Parental Control by AdguardHome
- Bandwith monitor
- Trafic monitor
- Telegram bot
- etc

## OpenWrt Image Information
- Default IP: 192.168.2.1
- Default username: root
- Default password: rtawrt
- Adguard Home IP: 192.168.2.1:8080
- Adguard home default username: admin
- Adguard home default username: naufaladmin
